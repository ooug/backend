# Backend

OOUG backend using javascript and express with mongodb
Hosted on Google Cloud Platform through App Engine.

# Setup

```
node version v14.x LTS

```

# Installation

```
  npm install
  npm run dev

```

# Environment

- copy .env.example file and paste as .env file update the config required.
