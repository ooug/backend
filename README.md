# Backend

ooug backend using typescript and express with mongodb

# Setup

```
node version v10.20.1 LTS
typescript version 3.9

```

# Installation

```
  npm i -g yarn
  yarn
  yarn local

```

# Environment

- copy .env.example file and paste as .env file update the config required.
